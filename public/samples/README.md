# Sample Images Directory

Place sample images here for the image generation feature. The application expects the following files:

- `portrait.jpg` - A sample portrait photo
- `landscape.jpg` - A sample landscape photo  
- `product.jpg` - A sample product photo
- `interior.jpg` - A sample interior/room photo

These images should be:
- JPEG format
- Less than 2MB each
- At least 512x512 pixels
- Appropriate for demonstration purposes

You can use royalty-free images from:
- https://unsplash.com
- https://pexels.com
- https://pixabay.com
